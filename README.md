# Python DSA Practice

A collection of Data Structures and Algorithms (DSA) problems solved in Python for coding interview preparation and problem-solving practice.

## Language

* Python 3

## Topics Covered

### Arrays

* Largest Element in an Array
* Second Largest Element in an Array
* Frequency Count
* Merge Two Arrays
* Missing Number
* Move Zeroes to End
* Remove Duplicates from Array
* Subset Sum
* Sum of Array Elements
* Two Sum
* Unique Element

### Strings

* Count Vowels in a String
* Character Frequency Count
* Palindrome String
* Reverse String
* String Length

### Number Theory & Math

* Armstrong Number
* Count Numbers Divisible by 3 or 5 (1 to N)
* Largest Number Less Than or Equal to N Divisible by K
* Palindrome Number
* Prime Number
* Sum of Odd Digits

### Pattern Problems

* Alpha Pyramid Pattern
* Full Pyramid Pattern
* Hollow Square Star Pattern

### Sorting Algorithms

* Merge Sort
